Para poder correr el programita debe tener instalado nodeJS en la pc, si no lo tiene puede descargarlo de 
la pagina oficial: https://nodejs.org/es/

Luego de instalarlo, debe ir a la carpeta raiz del programa y ejecutar el siguiente comando: 
npm i
El comando ejecutara las librerias necesarias para poder levantar el programa.

Al terminar las instalaciones debe colocar el comando <npm start index.js>, no debe colocar las flechas, 
este comando se encarga de ejecutar el programa en la consola.
